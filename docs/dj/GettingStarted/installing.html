Installing DataJoint¶

Installing DataJoint for Matlab¶

How do I install the Matlab DataJoint library?

Matlab specific prerequirements¶

  • MATLAB R2011a or higher. Earlier versions can be used with a few modifications.
  • mYm library used as the low-level MySQL API. The original project is at sourceforge and at Matlab Central.

Matlab install steps¶

  1. Download the latest matlab release of datajoint and unzip it.
  2. Place the unzipped folder somewhere where it won’t be deleted.
  3. Edit startup.m to add the datajoint directory to the search path and execute <datajoint-path>/mym/mySetup.m.
% startup.m
addpath <datajoint path>
run <datajoint-path>/mym/mymSetup

At the MATLAB command prompt, test that DataJoint and mym are on your path:

>> mym version
mYm version 2.6.8
>> which dj.conn
/<datajoint-path>/+dj/conn.m

To use DataJoint you need access to a MySQL server. See Configuring a Connection.

Installing DataJoint for Python¶

“How do I install the Python DataJoint library?”

Python specific prerequirements¶

DataJoint is written for python 3.4. For Mac OS X and Windows users, we recommend the anaconda distribution.

Additionally, you will need the following libraries

  • numpy
  • pymysql
  • matplotlib
  • json
  • pandas
  • networkx

Python Install Steps¶

Run

pip install git+https://github.com/datajoint/datajoint-python

Alternatively,

  1. Download the latest release of datajoint and unzip it.
  2. CD into the folder and run
python setup.py install

DataJoint is an Python/Matlab ORM, designed for busy scientists and collaborative data sharing.
